## HOW TO COMPLETE THE BRIEF

### Step 1 — Load Monday data (2 min)
Open the lead's Monday.com record (Board 5891626711). Pull:
- Name, address, bedrooms → PROPERTY section
- STR Profit column → STR net figure
- Long Term Let column → LTL figure
- Rent/Mortgage column → monthly cost
- Lead Profile column → profile notes
- Call Brief column → any previous conversation summary

### Step 2 — Load Drive intelligence entry (2 min)
If an entry exists (check Drive folder `1OTunhg09OSslWn0F_9Qy6fTX5ZMnAv1v` for their entry number):
- Pull key intelligence, objections, what resonated from previous calls
- Note any delay signals already identified
- Note language signals that worked

### Step 3 — Set the meeting goal (1 min)
Based on what you know:
- READY_NOW profile → goal is close/offer
- READY_SOON with known delay → goal is conditional offer or M2 with partner
- DELAYED → goal is qualify, confirm timeline, set specific follow-up
- Unknown → goal is qualify and assess

## QUICK REFERENCE — BRIEF BY PROFILE TYPE

| Profile | Key things to pre-load | Likely meeting goal |
|---------|------------------------|---------------------|
| STL-SW | Self-managing timeline, Airbnb listing details if known | Close if GL-SELF present |
| EX-STL | Reason they left, previous agreement terms | Close early — already sold on model |
| PURCH | Completion date, exchange status, property identified? | Deferred offer if exchanged |
| SELL | Sale timeline, asking price vs STR income gap | Income pitch must win vs sale |
| ABROAD | Departure date, country, remote owner comfort | Urgency close if imminent |
| R2R | Proposed rent-to-rent structure, economics at 13%+VAT | Assess viability, may not offer |

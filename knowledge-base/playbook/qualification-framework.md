# Stayful Lead Qualification Framework v3.0
*Built from 48 meeting transcripts | Last updated: 2026-06-07*

---

## CLASSIFICATION SYSTEM

Every lead is classified as one of:

| Class | Definition | Action |
|-------|------------|--------|
| **READY_NOW** | GL signal present, no unresolved DLY gate, property viable | Offer immediately. Set deadline. |
| **READY_SOON** | No gate or gate resolving within 6 weeks | Offer deferred to resolution. Book date. |
| **DELAYED** | Gate present, resolution timeline 2-6 months | Nurture with touchpoint. Don’t offer yet. |
| **NURTURE** | Viable but timeline unclear | Quarterly check-in. Keep warm. |
| **LOST** | Product mismatch, unresolvable gate, deadline passed | Close in CRM. Flag re-engage if profile strong. |

## GREEN LIGHT SIGNALS

### GL-SELF (Self-motivation signal)
Propery and owner circumstances create natural momentum:
- No mortgage (all income is profit)
- Owner has personal use need (STL enables blocking that LTL doesn’t)
- Renters Reform Act concern (legitimate fear of reduced landlord rights)
- Property already furnished and ready

**Conversion impact:** High. When present with no DLY gate: READY_NOW.

### GL-MGMT (Management dissatisfaction signal)
Owner currently with another STL agent and actively unhappy:
- Review count: <5 reviews in 6 months = critical
- Occupancy gap: >10pp below market = critical
- Maintenance failures mentioned
- Poor communication as explicit complaint

**Conversion impact:** Highest of all GL types. EX-STL with severe underperformance = near-certain conversion with right offer.

**Immediate tactic:** Ask “How many reviews have you had in the last 3 months?” in the first 5 minutes. Opens the performance gap.

### GL-TIME (Time pressure signal)
External deadline creating natural urgency:
- Tenant leaving within 6 weeks
- Keys received / receiving soon
- Relocation date approaching
- Off-plan completion within 8 weeks

**Conversion impact:** Medium to high. Only effective if offer deadline is set BEFORE the gate resolves.

## DELAY SIGNALS

| Type | Description | Conversion Rate | Strategy |
|------|-------------|-----------------|----------|
| DLY-TEN | Tenant in property | Low until vacant | Sign with deferred start. Don’t wait for vacancy to sign. |
| DLY-FUR | Property unfurnished | Medium | Sign now, furnish after. Stayful advises on spec. |
| DLY-PUR | Purchase not complete | Medium | Sign at exchange. Beat developer affiliate. |
| DLY-MOV | Moving abroad | High (urgency) | Relocation date = deadline. Move fast. |
| DLY-OTH | Spouse gate / renovation / competitor eval | Varies | See sub-type strategies. |

## PRE-QUALIFY QUESTIONS (first 2 minutes)

1. **What’s the current situation?** (LTL tenant / STL with another agent / just purchased / selling)
2. **When would you be looking to start?** (Identifies DLY type immediately)
3. **Is your partner involved in the decision?** (Spouse gate: get them on the same call)
4. **What income are you hoping for?** (If “guaranteed consistent income”: probe for guaranteed rent demand)
5. For R2R leads: **What is the R2R rent + bills total?** (Calculate margin viability before meeting)

## VIABILITY THRESHOLDS

| Property type | Minimum STL net | Notes |
|---------------|-----------------|-------|
| Standard owner-occupier | £12k/yr | Below this: STL advantage vs LTL is insufficient |
| R2R | £2k/yr margin after rent + bills | Below this: not viable for full management |
| 1-bed any type | Acceptable | But: no guaranteed rent product available |
| EX-STL with agent | Any | If underperforming, viability comes from displacement not absolute income |

## OFFER DECISION TREE

```
Is there a spouse/partner gate? → YES: No offer until they’re on the call
                                  → NO: continue

Is there a guaranteed rent demand + 1-bed? → YES: Pre-qualify out
                                            → NO: continue

Is the STL net viable (>£12k)? → NO: Redirect or decline
                               → YES: continue

Is there at least one GL signal? → YES: OFFER 13%+VAT
                                 → NO: Warm follow-up, no offer yet

Is the deadline set BEFORE the gating event? → YES: Send contract
                                              → NO: Recalculate deadline
```

| Profile | Conversion Priority | Lead Strategy |

| SELL bridge | MEDIUM | Frame as flexible income, not commitment. |

| STL-SW with spouse gate | LOW-MEDIUM | Get both on call before offering. |
